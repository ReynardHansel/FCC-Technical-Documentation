Introduction

Cascading Style Sheets (CSS) describes how HTML elements are to be displayed on screen, paper, or in other media. It saves a lot of work as it can control layouts and designs of multiple web pages at once.

External stylesheets are stored in css files (.css)

    Selectors

    Now that you have created the place where you can write CSS (whether it's a style tag or a new .css file), it's time to write and use CSS.

    CSS targets an HTML attribute, then tells it to display a certain way. So to use CSS, first you have to select/target the element you want to style. This is done by what's called a "Selector" in CSS

    Ex:

                    
    p {
        color: red;
        font-size 20px;
        font-style: italic;
    }

    Now all the <p> tags in the html file will have red color, 20px big, and displayed in italic.

    There's a lot of CSS selectors but the most common used ones are the class and id selectors. You do this by defining a class and id in the html tag (Ex: <div class="card" id="first-card"></div>). And then to target a class, put a . in front of the class name. To target an id, put a # in front of the id name.

    Ex:

                    
    .card {
        padding: 2em;
        margin: 2em auto;
        font-size: 1.3em;
    }

    #first-card {
        background-image: url (background.jpg)
    }
